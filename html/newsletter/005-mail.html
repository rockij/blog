vue, react 기반의 프로젝트를 진행하면서 퍼블리싱 측면에서 이런 프레임워크 프로젝트에 도움이 될 수 있는 기준이 될만한 사이트가 있으면 좋을 거 같다는 생각이 있었는데 이번에 회사를 입사하면서 실행에 옮겨 보았습니다.

리액트 퍼블리싱 사이트

메뉴 구성은 home, component, screen case, pub sheet, storybook 입니다.
- home은 사이트 설명과 작업 셋팅 등에 대한 설명
- component는 컴포넌트 작업을 퍼블리싱 측면에서 작업 상태에 대한 기준을 제시
- screen case는 컴포넌트를 활용해 프로젝트 화면 퍼블리싱 레퍼런스
- pub sheet는 퍼블리싱 화면 관리 시트 예시
- storybook은 컴포넌트에 대한 상세 설명 및 스펙

운영 방향은 프로젝트 진행하며 나온 디자인을 가지고 컴포넌트를 활용해 화면을 퍼블리싱하여 레퍼런스를 쌓아 실제 리액트 프로젝트 진행시에 작업에 도움이 되는 사이트를 만드는 것입니다.

현재의 레퍼런스는 예시로 몇개 정도 잡았지만 앞으로 프로젝트 진행하면서 꾸준히 업데이트를 하는것이 목표이며 업데이트시 해당 게시글에 공유할 예정입니다.
